# CS-Daily 🚀

A structured, accountability-driven journey through Go, CS fundamentals, and distributed systems.

## How it works

1. A challenge is generated with a reading resource + problem statement
2. Solve it, commit your code here
3. Julia (AI) reviews the code like a senior staff engineer
4. Address the feedback, iterate until it lands
5. Feedback shapes the next challenge

## Structure

```
challenges/          → one folder per challenge (YYYY-MM-DD_title/)
  completed/         → archived finished challenges
reviews/             → code review feedback files
agent/               → curriculum, progress tracking, cron spec
```

## Tracks

- **Track A:** Go language mastery
- **Track B:** CS fundamentals (algorithms + data structures)
- **Track C:** Distributed systems (HTTP → Postgres → Redis → Kafka → gRPC)
