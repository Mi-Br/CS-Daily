# plan/

This folder is our shared training plan. Both Michail and Julia edit it.

## Files

- **PLAN.md** — the big picture: upcoming weeks, what's working, what's hard, agreed rules
- **weeks/YYYY-Www.md** — one file per week: reading, exercises, project, Michail's notes

## How it works

1. Julia creates the week file with reading + project brief
2. Michail does the work, adds notes to the week file as he goes
3. Julia reads `PLAN.md` (especially "What Michail Finds Difficult") before creating the next week
4. Repeat

## Michail — how to use this

- Found something confusing? Add it to `PLAN.md` → "What Michail Finds Difficult"
- Had a thought mid-exercise? Drop it in the week file under "Michail's Notes"
- Want to revisit something? Note it in PLAN.md
- Want to change the pace or direction? Edit PLAN.md and tell Julia

This is your training plan. Own it.
